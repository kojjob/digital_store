# Changelog

All notable changes to the DigitalStore project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Session management page with device tracking
- Real-time form validation for registration and login
- Dark mode toggle in user settings
- Password strength indicators in registration form
- Enhanced security for session management
- Active session monitoring with device information
- Project documentation (DOCUMENTATION.md, GETTING_STARTED.md, CONTRIBUTING.md)

### Fixed
- Email validation on registration form
- Password visibility toggle behavior
- User profile image handling

## [0.1.0] - 2025-03-24

### Added
- Initial project setup with Ruby on Rails 8
- Basic user authentication using Devise
- TailwindCSS integration for styling
- Hotwire (Turbo and Stimulus) for interactive features
- Landing page with product showcase
- PostgreSQL database configuration
- User registration and profile management
- Responsive design with mobile navigation
- Basic help and contact pages
- Docker configuration for deployment

### Changed
- Updated Rails from 7.x to 8.0.2
- Switched from Webpacker to Import Maps for JavaScript management

### Fixed
- Initial database migration issues
- User authentication edge cases
